PROJECTS

Wireframes:

At the start of the course, the very first assessment we undertook was to design a website utlizing Wireframes to create the layout and design. By articulating the purpose of the website, I was able to generate ideas for the website and piece together concepts with aspects that I felt were in important. Clarity, Intuitive Navigation, and Visual Heirachy were prioritised. From here I was able to design a suitable site map, defining key screens, followed by a flowchart for how the pages would connect/interact. Through appropriate spacing, proportionate sizing, and soft colours, I was able to achieve 'Clarity'. Given that clarity is not only evoked visually but also through interaction, 'Intuitive Navigation' added to this via keeping the website structure and flow traditional/familiar with clear navigation points and simple relationships between pages. Since a website users are predominantly people, if we don't take from our life experiences and understanding of people, we cannot create a good website.

Portfolio:

Building off the previous Wireframes project, the website that was designed was built using strictly HTML and CSS. With an emphasis on Semantic HTML, a simple five page website with extensive CSS was created. Notable aspects demonstrating use of said CSS are the Transitions when hovering and clicking on links and image links, as well as the Modal Images exhibited as the 'About' and 'Skills' pages, where use of filters being applied can be seen.
